# 🎮 2025 대선 시뮬레이터

미연시(비주얼 노벨) 스타일로 2025년 대선 후보들을 만나보는 인터랙티브 웹 게임입니다.
꿈속 세계에서 천사 고양이와 함께 각 후보의 방을 둘러보며 정책을 알아가세요!

## ✨ 게임 특징

### 🌙 스토리 기반 게임플레이
- **타이틀 화면**: 아름다운 배경 이미지와 함께 시작
- **거실 씬**: TV 뉴스를 보다 잠들어가는 5단계 스토리
- **꿈속 세계**: 천사 고양이가 안내하는 환상적인 공간
- **후보 방 탐험**: 각 후보의 개성 있는 방에서 정책 탐색
- **엔딩**: 모든 후보를 만난 후 현실로 돌아가는 마무리

### 🎵 몰입감 있는 사운드
- **BGM 시스템**: 모든 화면에서 일관된 배경음악
- **음량 조절**: 30% 볼륨으로 최적화된 사운드
- **토글 기능**: 언제든 음악 on/off 가능

### ⏰ 실시간 선거 정보
- **D-Day 카운터**: 2025년 6월 3일까지 실시간 카운트다운
- **정확한 시간**: 일/시/분/초 단위로 정밀 표시
- **모든 화면 동기화**: 게임 전체에서 일관된 정보 제공

## 🎨 디자인 & UI

### 레트로 픽셀 아트 스타일
- **각진 네모 디자인**: 모든 UI 요소가 픽셀 게임 스타일
- **일관된 테마**: 둥근 모서리 없는 레트로 감성
- **정당별 색상**:
  - 🔵 이재명 (민주당) - 파란색
  - 🔴 김문수 (국민의힘) - 빨간색
  - 🟠 이준석 (개혁신당) - 주황색
  - 🟡 권영국 (진보당) - 노란색

### 모바일 최적화
- **고정 해상도**: 390×844px (iPhone 14 기준)
- **중앙 정렬**: 웹에서 모바일 게임처럼 표시
- **터치 친화적**: 큰 버튼과 직관적인 인터페이스

## 🎯 게임 플로우

```
1. 타이틀 화면 → 시작하기 버튼 클릭
2. 거실 씬 → 5개 대사 진행 (터치로 넘김)
3. 꿈속 세계 → 천사 고양이와 대화 후 후보 선택
4. 후보 방들 → 각 방의 오브젝트 클릭으로 정책 확인
5. 엔딩 씬 → 게임 완료 축하 및 선관위 사이트 연결
```

## 🏠 후보별 방 특징

### 이재명 방 (특별 구현)
- **실제 이미지 오브젝트**: 화분, 컴퓨터, 아파트, 서류봉투
- **상세 정책 정보**: 각 오브젝트별 구체적인 공약 내용
- **크기 조정**: 컴퓨터 1.5배 크기로 중앙 배치

### 다른 후보 방들
- **픽셀 아트 오브젝트**: 책장, 소파, 책상, 화분
- **정책 다이얼로그**: 클릭 시 해당 후보의 정책 정보 표시
- **개별 배경**: 각 후보별 고유한 방 배경 이미지

## 🛠️ 기술 스택

### Frontend
- **Flutter Web**: 크로스 플랫폼 웹 개발
- **Dart**: 프로그래밍 언어
- **HTML5 Audio API**: 웹 BGM 구현

### 상태 관리
- **Stream 기반**: BGM 상태 실시간 동기화
- **StatefulWidget**: 게임 상태 관리
- **Navigator**: 화면 전환 및 라우팅

### Assets & Media
- **배경 이미지**: intro_background.png, livingroom_background.png, angelcat_background.png
- **후보별 이미지**: 방 배경, 아바타, 정책 오브젝트 이미지
- **BGM 파일**: assets/bgm/ 디렉토리

## 🚀 시작하기

### 필요 조건
- Flutter SDK (3.0.0 이상)
- Chrome 브라우저 (웹 실행용)

### 설치 및 실행
```bash
# 저장소 클론
git clone https://github.com/jangjia01234/ez-vote-kr.git

# 프로젝트 디렉토리로 이동
cd politics-web-flutter

# 의존성 설치
flutter pub get

# 웹에서 실행
flutter run -d web-server --web-port=5000
```

### 브라우저에서 확인
```
http://localhost:5000
```

## 🎮 게임 조작법

### 기본 조작
- **터치/클릭**: 대사 넘기기, 버튼 선택
- **BGM 버튼**: 좌상단에서 음악 on/off
- **뒤로가기**: 후보 방에서 좌상단 화살표 버튼

### 게임 진행
1. **타이틀**: "시작하기" 버튼으로 게임 시작
2. **거실**: 화면 터치로 대사 진행
3. **꿈속 세계**: 후보 버튼 클릭으로 방 선택
4. **후보 방**: 오브젝트 클릭으로 정책 확인
5. **엔딩**: 선관위 사이트 방문 또는 게임 재시작
